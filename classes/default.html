    Runs a callback before saving an Item.

    Parameters

    • item: ItemPayload

      An existing Item to be saved.

    • presave: any

      Allows clients to perform any actions last second before the save actually occurs (like setting previews). Saves debounce by default, so if a client needs to compute a property on an item before saving, it's best to hook into the debounce cycle so that clients don't have to implement their own debouncing.
